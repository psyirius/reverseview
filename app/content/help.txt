<div class="generalheading2">Getting started with VerseVIEW</div>

<div class="generaltext"><p>VerseVIEW is a software that will allow displaying Bible verses and Song lyrics in a presentation format.
Additional information are available on the VerseVIEW website.</p></div><br>

<div class="generalheading2" style="font-weight: normal;">Tabs</div>

<div class="generaltext">

  <p><u>Bible</u> - This tab contains controls for selecting a specific verse and presenting it.This includes
    quick reference text boxes, recently selected verses, word search, buttons for navigating verses and scheduling. </p><br>

  <p><u>Songs</u> - This tab contains controls for selecting a songs and presenting it. This includes
    song list with pages, lyrics search, buttons for navigating verses and scheduling. </p><br>

  <p><u>Verses</u> - This tab lists the verses of the selected chapter. Also provides an icon for notes.</p><br>

  <p><u>Lyrics</u> - This tab lists the lyrics of the selected song.</p><br>

  <p><u>Notes</u> - This tab is for reviewing the notes taken for the selected chapter.</p><br>

  <p><u>Search</u> - This tab displays the search results and parameters for advanced search.</p><br>

  <p><u>Schedule</u> - This tab displays controls to manage schedule.</p><br>

  <p><u>Graphics</u> - This tab is for selecting the foreground color and background images.</p><br>

  <p><u>Screen</u> - This tab is for configuring presentation and stageview screens.</p><br>

</div>
